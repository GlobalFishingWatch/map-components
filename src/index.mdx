# Set of UI components for Global Fishing Watch.

## Installation

```bash
yarn add @globalfishingwatch/map-components
```

## Peer dependencies needed

```bash
yarn add dayjs react-spring
```

## Usage

```jsx
import { YourComponent } from '@globalfishingwatch/map-components'

class MyApp extends Component {
  render() {
    return <YourComponent />
  }
}
```
