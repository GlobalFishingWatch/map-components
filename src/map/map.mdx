# Map

## Description

This component displays and load fishing activity and fishing-related layers on a map, used in various GFW projects:

- [Map client](https://github.com/GlobalFishingWatch/map-client/blob/9c5179358d4870c0fd4c2c0742917e72e6025c59/app/src/map/components/MapWrapper.jsx#L5)
